CineExplorer is a cutting-edge movie discovery app designed for Android.
Leveraging the power of the TMDB API, it offers users a sleek, modern interface to explore trending movies and TV shows, watch trailers, and find where to stream content in their country.

Key Features:
- Real-time Trending Lists (Day/Week)
- Detailed Movie & TV Show Information (Cast, Seasons, Similar Titles)
- Official Trailer Playback
- Dynamic Streaming Provider Search (Netflix, Disney+, etc.)
- "Where to Watch" filtered by your specific region
- Advanced Filtering (Genre, Year, Rating, Runtime)
- Beautiful Dark Mode UI with smooth animations

Built with modern web technologies (React, TypeScript, Tailwind CSS) and compiled natively for Android using Capacitor.
